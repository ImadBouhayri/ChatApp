Hello,

First Download the source code and run it in android studio.

Main Page:List of 200 random users,each user is composed of his name, 
	  picture, last message sent, time, listed in descending order.
          When a user is clicked go to second page.

Chat page:You can send a message to the user clicked by clicking in the text 
field in bottom, the keyboard will appear, right your message and click
the Send button, the message will be sent the user in the title will reply 
back with the same message in 0.5 seconds,the back button in the top right
is to return back to the list of users,shows all chats that have been sent 
back and forth previously in the content layout.
